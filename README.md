#Lekoloane Rina Portfolio Website

This is a personal portfolio website built to showcase Rina’s **professional profile, projects, blog content, and petitions** in a structured and user-friendly way.

The platform serves as a central digital space where visitors can learn about her background, explore her work, read articles, and engage with causes she supports.

---

## 🌐 Features

- **Portfolio Section**  
  Showcases projects, skills, and experience.

- **CV / Resume**  
  Displays professional background, education, and achievements.

- **Blog**  
  A space for sharing insights, ideas, and written content.

- **Petitions**  
  Allows users to view and engage with important causes.

- **Responsive Design**  
  Optimized for desktop and mobile devices.

---

## Tech Stack

**Frontend**
- HTML, CSS, JavaScript  
- React

**Backend**
- Node.js / Express
